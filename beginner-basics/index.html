Beginner Basics — Start Here Before Session 1

This folder is the absolute-beginner on-ramp to the main 20-hour plan. If you've never written JavaScript, never used a terminal, or never installed Node, do these 5 mini-lessons first. Each is short (30–60 min) and assumes zero prior knowledge.

Learn Node.js/ Beginner Basics/Beginner Basics — Start Here Before Session 1

Total time: ~6 hours (one-time investment, spread over a few days).

Why a separate folder? Why it matters

The main plan (session-01 … session-10) assumes you already know:

  • How to open a terminal and run commands
  • Basic JavaScript (variables, functions, arrays, objects)
  • What a "file path" and "folder" mean in code
  • How to install software
  • How to read an error message and use Git

If any of that feels shaky — start here. Otherwise jump straight to ../session-01-fundamentals.md.

How to use these lessons Concept

  • Read top to bottom. Type the code yourself — don't copy/paste.
  • If something doesn't work, re-read the previous step.
  • End each lesson with the 5-minute recap at the bottom.
  • It's OK to be slow. Understanding > speed.
