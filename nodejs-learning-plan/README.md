# Node.js in 20 Hours — Pareto (80/20) Learning Plan

A focused 20-hour roadmap covering the **20% of Node.js concepts that deliver 80% of real-world results**. Designed as **10 × 2-hour sessions**, each ending with a **15-minute review**.

> 🟢 **Brand new to programming or JavaScript?** Start in [beginner-basics/](beginner-basics/README.md) (~6 hours — JS, terminal, Git).
> 🔵 **Know JS but new to Node?** Do [nodejs-basics/](nodejs-basics/README.md) next (~8 hours — Node concepts explained slowly).
> Both are optional warm-ups. When ready, come back here for Session 1.

---

## How to Use This Plan

- **Pace:** 1 session per day = 10 days. Or 2 per day = 5 days (intense).
- **Format of each session (2h = 120 min):**
  - 0–15 min  → Warm-up & concept overview (read/watch)
  - 15–90 min → Hands-on coding (the core)
  - 90–105 min → Mini-exercise / challenge
  - 105–120 min → **15-min Review** (recap notes, self-quiz, commit code)
- **Setup before Session 1:**
  - Install [Node.js LTS](https://nodejs.org)
  - Install [VS Code](https://code.visualstudio.com)
  - Verify: `node -v` and `npm -v`
  - Create a folder `node-learning/` with one subfolder per session.
